## Tested on

* Linux
  - Linux Mint 20
  - Node 10.19.0

## Quickstart

```sh
git clone https://github.com/hotteshen/eastec.sone.piapp.git
cd eastec.sone.piapp
npm install
npx electron-rebuild
npm start
```

## Build

```sh
npx electron-builder
```

Look inside of `dist/` directory.
